## 1. FrontEnd - Angular

Inicializar projeto utilizando Angular 19 + Po-ui, tendo a seguinte estrutura:

- Sistema de login, utilizando api mockada

- Menu utilizando https://po-ui.io/documentation/po-menu, tendo as opções:

  1) Cadastro de Clientes - Rota clientes
  
  2) Cadastro de Produtos - Rota produtos
  
  3) Cadastro de Contratos - Rota contratos
  
- A cada rota do menu, criar listagem utilizando o componente https://po-ui.io/documentation/po-page-dynamic-table


- Em cada listagem, criar o botão incluir utilizando o componente em https://github.com/poui-mobile/po-ui-json-forms
